Data Carousel dashboard:

This page summarizes input datasets staging from TAPE RSE.

By default page shows all non-finished stagings and staged datasets that were modified in last 24 hours. To change the time range of the selection, use the Change button in the top sub-bar. To apply the new time range push the Apply button.

Use multi-select menus in top-right corner of the page to select particular campaign or source RSE. Click on the Update button to reload the data.

To look for potentially stuck files click on sign in Rucio rule column which is available for datasets in active staging state only. The new table will appear below with list of failed file stagings. Note, all failed attempts are shown so number of rows in the table != number of stuck files. We are limited by 1000 of rows because of the peculiarity of the data source.
