Help:

Contextual help for monitor pages can be found at the bottom of each page. Use the help menu in the upper right corner to navigate directly. All the monitor-internal help documentation is gathered on a complete help page available in the Help menu at upper right.

Bookmarking pages of interest to you is recommended. Save that useful drill-down rather than doing all the clicking again. All monitor pages are bookmarkable (the full specification of the page is in the URL). The most relevant links based on your visit history can be found on the top of My BigPanDA page. Not all the capability is expressed explicitly on the interface at present (partly deliberately to go easy on the DB). You can often extend the depth and breadth of queries with options like &days=N, &limit=N on the URL.

{% comment %}

The monitor has a secure https mode. If you go to https://bigpanda.cern.ch and you have configured your browser to present your grid certificate, the monitor will automatically recognize and authenticate you if you have run PanDA jobs with that cert. This authentication will enable you to access non-public information in the monitor and invoke actions through the interface (once available).

{% endcomment %}

If you need to use PanDA data programmatically, json dumps are available. See API section or search on 'curl' in the in the complete help page. Use with caution; do not pull huge amounts of data, especially systematically. Do not use web scraping, use these json dumps. If you need to extract other information, tell us.

Send your suggestions and requests for the PanDA monitor to PanDA monitor support.
